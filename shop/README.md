# Plugin `shop`

Compagnon du cours `academy.ottho.co/claude-shopify`. Bootstrap d'une boutique Shopify de A à Z avec Claude Code.

## Slash command

- `/shop:bootstrap` — pilote le workflow 11 phases (interactif par défaut, `phase=N` ou `audit` en argument)

## Skill

- `bootstrap` — workflow détaillé + scripts + templates. Loadé automatiquement par la commande.

Voir `skills/bootstrap/SKILL.md` pour le détail technique.
